Eclipse快捷键

基本Eclipse的快捷键（有部分是通用快捷键)

查看快捷键：

编辑

  • Junit 运行：Alt + Shift + e，再t。
  • 注释或取消注释：Ctrl + /。
  • 块注释：Ctrl + Shift + /；或选定多行后用同上。
  • 上一步：Ctrl + z。
  • 下一步：Ctrl + y。
  • 代码提示：Alt + /。
  • 删除一行：Ctrl + d。
  • 复制本行到下或上：Ctrl + Alt + 向上（向下）。
  • 下方新建空白行：Shift + Enter。
  • 本行内容上下移动：Alt + 向上（向下）。
  • 上次编辑的位置：Ctrl + q。
  • 下一个单词：Ctrl + 向右。
  • 快速导入：Ctrl + Shift + o。
  • 快速定位到行：Ctrl + i。
  • 快速对齐：Ctrl + Shift + f。
  • 查找被谁调用：Ctrl + Shift +g。（或选词后右键reference -> workspace）

界面

  • 整个eclipse界面最小化：Alt + space再选。
  • 快速outline：Ctrl + o。
  • 切换编辑页面：Ctrl + PageUp(PageDown)。
  • 上下移动页面：Ctrl + 向上（向下）。
  • 页面顶部底部：Ctrl + Home（End）。
  • 大屏幕显示编辑器：Ctrl + m。

其他

  • outline里图标含义：绿色指公有，红色指私有，空心指变量，实心指函数。
  • 编辑器分屏：左边编辑页面拖动直到有分界栏显示即可。
  • 恢复界面perspective：打开Windows -> perspective -> resetperspective。
  • 下载api：https://pan.baidu.com/s/1e7EOchsrNLE2NPJG-iIRcw 密码：vlld 或pan_艾_计算机软件)。
  • 同名变量高亮显示：打开Windows-> preferences-> java-> Editor-> Mark Occurences。
  • Eclipse里查看源码（安装）：
  1. 点 “window-> “Preferences” -> “Java” -> “Installed JRES”。
  2. 选择你要添加源码的jre，点击右边edit，编辑。
  3. 进入JREDefinition页面，选择你安装jre路径里面的lib文件夹里面的rt.jar。选择后点击右边的Source Attachment:(none)。然后选择您下载JDK文件夹下面的”src.zip”文件就可以了。【1】
