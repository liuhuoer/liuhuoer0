安装python2和python3

在同一系统中使用python2及python3。（linux下未补充）

Windows下

安装python2和python3

链接：

  • python3.5.2
  • python2.7.9

配置环境变量

  • （假定python2安装目录为D:\Python\python27,配置python3类似）
  • D:\Python\python27
  • D:\Python\python27\Scripts

找到python3 IDLE安装位置，发送快捷方式到桌面

修改python.exe名字

  • 修改python2.7.9和python3.5.2安装目录中的 python.exe和pythonw.exe的名称为python2.exe、pythonw2.exe和python3.exe、pythonw3.exe。
  • （cmd命令，输入python2即可运行python2.7.9版本，python3即可运行python3.5.2鞍姹荆?

pip2 和 pip3设置

python安装包需要用到包管理工具pip，由pip安装的包的位置（根据个人安装）目录如下：D:\Python\Python37\Lib\site-packages

  • 在DOS命令框输入命令，python3 -m pip install –upgrade pip –force-reinstall，显示重新安装成功。

  • 在DOS命令框输入命令，python2 -m pip install –upgrade pip –force-reinstall，显示重新安装成功。【1】
